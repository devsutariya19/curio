# Basic Authentication

## Objective
Allow users to sign in and out using Supabase Auth.

## Implementation
- **Service**: Supabase Auth
- **Details**: We will use the Supabase client library to handle user authentication. This will include sign-up, sign-in, and sign-out functionality. We will primarily use email/password authentication for the MVP.
