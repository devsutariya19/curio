# Getting Started with CurioDocs

Welcome to CurioDocs, a modern documentation platform built for teams who value collaboration and beautiful content. This guide will help you get up and running quickly.

## What is CurioDocs?

CurioDocs is a collaborative documentation platform that combines the simplicity of Markdown with the power of real-time collaboration. Built with Next.js and Supabase, it offers:

- **Markdown-first approach** with MDX support
- **Real-time collaboration** for teams
- **Role-based permissions** for secure access
- **Version control** to track changes
- **Full-text search** across all content

## Quick Overview

Before diving into the details, here's what you can expect:

1. **Write** documentation in Markdown
2. **Collaborate** with your team in real-time
3. **Organize** content with folders and categories
4. **Share** knowledge with granular permissions
5. **Track** changes with built-in version control
